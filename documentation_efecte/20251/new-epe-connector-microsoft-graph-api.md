As an admin of Efecte's Provisioning Engine tool (EPE), I want to do provisioning / deprovisioning with Microsoft Graph API.
Background information:
What?

 New Microsoft Graph API connector to EPE

How?

 Administrators are able to read (scheduled) and write (event-based) data with Graph API in EPE UI. 
  
   Processes can be scheduled in ESM Provisioning Tasks UI
   Processes can be run manually from ESM Provisioning UI
   Processes can be run event-based triggered by Visual Workflow Automation
  
 Provisioning Engine (EPE) is able to read (scheduled) devices info from Intune with Graph API
 Provisioning Engine (EPE) must be able to write (event-based), remove device by using wipe or retire actions with Graph API
  
   Processes can be run event-based triggered by Visual Workflow Automation
  

Why?

 EPE connector catalog must be comprehensive for customers
 Graph API and Intune support has been requested by several customers

To whom?

 EPE Administrators
 EPE Customers
