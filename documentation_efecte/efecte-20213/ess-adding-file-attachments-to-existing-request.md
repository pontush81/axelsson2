 User story:  
 As a Self-Service Portal user, I want to upload additional file attachments to existing request using my open requests' commenting.  
  
 AC:  
 
 Uploading the additional files to open requests shall be possible by using commenting of my open requests 
 Multiple files can be added to one comment 
 File name with file extension shall be included in the comment that is sent to the Service Management tool worklog -handler. 
 Maximum file size shall be the same as set in the settings area
