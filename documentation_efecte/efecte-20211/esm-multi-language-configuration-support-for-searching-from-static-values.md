ContentsUser story:ACs:User story: 
 As a system administrator, I want to be able to search attribute static values for different languages in order to find relevant data of my own language. As an end-user, I want that bookmarks created in different languages find correct results.  
ACs: 
 
 User shall be able to search data cards by using its own language of translated static strings in global search 
 User shall be able to search data cards by using its own language of translated static strings in conditions search 
   
   'like' -condition shall match to a given value of every language 
    
 User shall be able to search data cards by using its own language of translated static strings in filtering 
 Bookmarks, search templates, and public views are shown to the end-user with his/her own language even if the user that created the item had a different language 
 EQL queries (search conditions and search results) shall work on system language
