To address issues found in the latest version, we have made a new 2025.3 service release to be available as an opt-in update for customers actively impacted by the issues below.
Why opt-in? To ensure minimal disruption, we’re offering this release only to customers who need these fixes today. This allows us to address your specific challenges more quickly while maintaining stability for rest of the customers.
How to get the update: If these fixes resolve issues in your environment, simply contact our service desk to schedule the upgrade for version 2025.3.5-SR-1. Note: These fixes and improvement will also be included in our next standard release for all customers.
 
Release notes:
Service management tool 2025.3.2

 529705 Show more emails in GraphAPI email view: BETA GraphAPI email handler UI now supports loading more emails if there are more than five emails. Emails can be marked read or unread.
 530420 CP-8361: Improved report loading times in Agent UI: Graphical views performance has been improved and dashboard times to load all cells have been also reduced. This removes the ability of the bar and line charts to show animated load.
 530990 CP-10743: Agent UI Email window is not closed after send button is pressed: Email popup now closes when email is successfully sent and email is shown in the UI immediately. Error messages are displayed correctly if the email sending fails.
 531841 CP-10731: Text attributes with rich text enabled - Link issues in Classic UI: Fixed rich text attribute display issue caused by incorrect string method in HTML sanitization.
 532063 CP-10737: Bug with ReferenceCopy handler and static value as a checkbox: Fixed static values not being copyable if target static attribute has showAsCheckboxes enabled.

Native connectors

 IGA-10281 New task history UI for scheduled tasks: Native Connectors scheduled task history UI and UX has been improved. New UI contains up-to 500 latest runs history. And better visibility to task run information.

Secure Access

 IGA-10948 Security improvement to Finnish Trust Network (FTN) authentication: Secure Access is now compliance with Traficoms newest requirements, related to FTN Bank authentication.

Full release notes are also available in the respective sections of the affected products in Docs: https://docs.efecte.com/20253
