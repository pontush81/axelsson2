 Problem statement  
 Administrators want to add titles and text blocks between the form fields to guide end users when filling out issues on the ESS2 user interface.    
  Short description  
 Administrators can add titles and text blocks to forms when configuring them.    
  Use case details  
 As an admin user, I want to include section titles on a form, to provide more guidance to an end-user completing a form.   
 AC:    
 
 When creating or managing a form, I can select to add a section title 
 When creating or managing a form, I can select to add a text block 
 I can arrange the order of the section title and text block. 
 I can see the section title and text block on the form in the self-service portal in the position specified 
 Section title and text block are only visible when creating an incident
