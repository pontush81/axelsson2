Problem statement
Efecte’s self-service portal (ESS) doesn’t meet A- and AA-level accessibility guidelines that are defined in WCAG 2.1. 
Short description
ESS shall meet WCAG 2.1 A- and AA-level guidelines so that Efecte’s portal can be offered to even larger user groups than have been before.
Use case details
Based on the accessibility audits, the following topics shall be improved to have better coverage in the area of software accessibility
AC:

 Improved screen reader support
 Keyboard accessibility improvements
 Label improvements
 Focus order improvements
 Contrast improvements for the Accessibility mode
 Several search accessibility improvements
