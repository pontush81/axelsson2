 Problem statement  
 Administrators want to map reference drop-down form fields to a single text attribute.    
  Short description  
 In the Self-service configuration area, the administrator can create a reference drop-down form field and map it to the field to the text attribute. The admin is informed while configuring if the text attribute is already used in the multi-mapping configuration.   
  Use case details  
 As an ESS2 admin, I want to configure reference drop-down field and map that to existing text field in Form configuration area.  
 AC:    
 
   Admin selects the reference drop-down field to the form   
   Admin can set the target attribute to a existing text attribute in target template
