 User story: As a customer I want to do event-based provisioning to IBM LDAP.  
  ACs:  
 
 Efecte Provisioning Engine (EPE) must be able to write (event-based) data to IBM LDAP 
 Administrators are able configure connection to IBM LDAP using the admin UI 
   
   Processes can be scheduled in ESM ProvisioningTasks UI 
   Processes can be run manually from ESM Provisioining UI 
   Processes can be run event-based triggered by Visual Workflow Automation 
    
 
 Notes: Following event-based provisioning capabilities will be available in the first version:  
 
 Verify User 
 Create User 
 Update User 
 Change User DN 
 Remove User Attribute 
 Remove User 
 Verify Group Membership 
 Add User to Group/Remove User from Group 
 Verify Group 
 Create Group 
 Update Group 
 Delete Group
