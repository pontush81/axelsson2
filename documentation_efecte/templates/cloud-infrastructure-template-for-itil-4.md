    CloudInfrastructure template.xml
  
  
   The attached XML.file includes an example Cloud Infrastructure template in accordance to ITIL 4. If the template is imported, then it will be added to the CMDB module of the Service Management Tool.  
 Remember to create folders to store data cards for this template. You also need to specify template and folder permissions. Also don't forget to set the references in this template because references are not imported. Finally, remember to remove and add the shared class of General Information according to your configuration.
