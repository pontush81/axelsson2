    Matrix42 Baseline Release Notes 2024.2
  
  
  Matrix42 Baseline 2024.2 includes several feature improvements and bug fixes across solution portfolio. 
The main improvements that apply to the entire baseline:

 New supported language: French
 Out commented blocks cleaned from ESS connector files
 new parameters added into organization connector

In Matrix42 Core & Pro Baseline main improvements are:

 Change management practises configuration in overall.
 Streamlining task set -feature

In IGA Baseline main improvements are:

 Departing user process
 Mass update entitlements and business roles to users
