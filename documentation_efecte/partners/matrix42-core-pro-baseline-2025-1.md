  Matrix42 Core & Pro Baseline main improvements
The latest update significantly enhances the Problem Management practice with optimized workflows, improved template configurations, and refined transformation functionalities. Users will benefit from streamlined problem identification and resolution processes, making management more efficient and effective.
This release introduces new features like the password reset option on tickets, improved configuration item (CI) relationships and launches the Customer Service Management (CSM) solution, expanding service capabilities. Additionally, there are notable improvements to baseline connectors and templates, providing better integration and support, particularly for Firescope DDM and Microsoft Intune users.
Overall, the update aims to enhance problem management processes while expanding service and device management capabilities, offering users greater efficiency and functionality.
IGA Baseline main improvements The latest IGA Baseline Release delivers essential improvements to access review processes, automation and reporting. It strengthens the foundations for secure and efficient identity governance while addressing known issues and enhancing usability across core functionalities. These updates improve accuracy, streamline processes, and support better compliance and user experience for both administrators and end users.
