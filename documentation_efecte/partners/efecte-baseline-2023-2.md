    Quick guide for Efecte Baseline 2023.2
  
  
  
    Efecte Baseline Release Notes 2023.2
  
  
  Efecte Baseline 2023.2 includes a significant number of improvements to both, IGA and ITSM solution. Also, Norwegian (Bokmål) is available as a new language.
IGA solution baseline's main improvements are:

 Handling users with the same name
 Access right management, including removing manual entitlements via Self-service portal
 Departing user process renewal

ITSM solution baseline's main improvements are:

 Streamlining the Incident and Service request Management confuguration. These processes have been transfomed to use same "Ticket" template instead of separate “Incident” and “Service request” templates, which have been used in former Efecte baseline versions.
 Content for automatic e-mail notifications maintained on datacards (Ticket, Task and Knowledge base article -templates), supporting multilanguage.
 ESS2 included
 Templates for different device types in Asset Management have also been transformed to use the same "Device" template.
 Process for creating local ESM users (single users with Ticket, mass import of users with CSVImportTask)

“Efecte Delivery Management” solution is included in release, offering streamlined way to deliver ITSM and IGA projects. Baseline includes predefined delivery tasks to be fulfilled which can be assigned to project stakeholders.
