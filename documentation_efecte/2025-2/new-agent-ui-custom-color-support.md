Problem statement
Having customization options allow organizations to set a familiar look-and-feel of to a user interface. Agent UI does not yet support custom colors.
 
Short description
Custom colors are now supported in Agent UI
 
Use case details

 Color customizations are supported now in Agent UI
 Following settings are used in Agent UI as-is (there is no need to configure them separately for Agent UI) 
  
   Header settings (Logo indentation and search alignment settings, background, button and text colors)
   Navigation settings (highlight, background, icon, text and button colors)
   Main view (button, icon, background and link colors)
  
 Some UI elements have a fixed color and cannot be changed
 Dark mode does not support custom colors
