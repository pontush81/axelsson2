As an admin of Provisioning Engine tool (EPE), I want to do provisioning / deprovisioning to Generic REST API
Background information:
What?

 New Generic REST API connector to EPE Connectors

How?

 Administrators are able configure Generic REST API using the EPE admin UI 
  
   Processes can be scheduled in ESM Provisioning Tasks UI
   Processes can be run manually from ESM Provisioning UI
   Processes can be run event-based triggered by Visual Workflow Automation
  

Why?

 Connectors catalog must be comprehensive for customers

To whom?

 EPE Administrators
 EPE Customers
