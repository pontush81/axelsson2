As an admin of Efecte's Provisioning Engine tool (EPE), I want to do provisioning / deprovisioning to Google directory.
Background information:
What?

 New Google Directory connector to EPE

How?

 Administrators are able configure connection to Google Directory using the EPE admin UI 
  
   Processes can be scheduled in ESM Provisioning Tasks UI
   Processes can be run manually from ESM Provisioning UI
   Processes can be run event-based triggered by Visual Workflow Automation
  

Why?

 EPE connector catalog must be comprehensive for customers
 This Google Directory support has been requested by several customers

To whom?

 EPE Administrators
 EPE Customers
