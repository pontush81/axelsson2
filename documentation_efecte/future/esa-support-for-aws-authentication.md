 As a customer of Efecte's Secure Access tool (ESA), I want to use Amazon Web Service authentication with SSO.   
Background information: 
What? 
 
 Amazon Web Services with SSO is asked by customers.  
 
How? 
 
   ESA is to be able to login users from Amazon Web Services with SSO   
 
Why? 
 
 ESA Authentication catalog must be comprehensive for customers 
 Manually entering the username and password for ESA is time-consuming and unproductive. SSO streamlines the user validation process for login and makes it easier to access protected resources. 
 
To whom? 
 
 All ESA customers 
   
   ESA Administrator can decide to use or not to use
