As an admin of Efecte's Provisioning Engine tool (EPE), I want to do provisioning / deprovisioning to Microsoft Teams.
Background information:
What?

 New Connector Microsoft Teams can read team info and its members from teams, comments and statuses

How?

 Administrators are able configure connection to Microsoft Teams using the EPE admin UI 
  
   Processes can be scheduled in ESM Provisioning Tasks UI  
   Processes can be run manually from ESM Provisioning UI  
   Processes can be run event-based triggered by Visual Workflow Automation, following  activities available for this: 
    
     Create Team
     Update Team
     Delete Team
     Add members to Team
     Remove members from the Team
     Send comment to the Team
     Send status to the Team
    
  

Why?

 EPE connector catalog must be comprehensive for customers

To whom?

 EPE Administrators
 EPE Customers
