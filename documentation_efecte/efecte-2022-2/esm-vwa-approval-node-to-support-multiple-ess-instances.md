 Our customers need to be able to configure ESS approvals to happen in a configuration where the ESM is connected with multiple ESS instances.  
 For this reason we will introduce new optional parameters to the approval node in Visual Workflow Automation. The parameters are "Approval portal token" and "Approval portal URL".
