   Efecte Secure Access login window has been redesigned. The video above shows you a refreshed view about login into Efecte with ESA.   
 The new login window provides more information to the user about the available login methods. The login options are organised in separated tabs according to the authentication method. If one of the authentication methods is not enabled, the associated tab is also not displayed.
