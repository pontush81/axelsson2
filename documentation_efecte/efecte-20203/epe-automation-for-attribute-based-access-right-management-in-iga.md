 User story: As a IGA manager I want to assign access rights based on user attributes. These could be for example title, location or cost center. This would add automation for new employees on-boarding process or for existing user lifecycle management, when they have changes in their work role.  
  ACs:  
 
 Efecte IGA can automatically assign access rights based on attributes when Human Recources Material as master-data 
 These access rights are used as a set of "basic access right" 
 If there are changes for chosen user attributes new set of "basic access rights" will be provisioned to user
