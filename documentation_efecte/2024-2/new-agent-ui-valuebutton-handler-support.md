ValueButton has not been supported in the new Agent UI. ValueButton handler would allow easy interaction with data cards directly in the list view and data card view, without the need to edit all details.
 
Short description
ValueButton handler is used to set a static value in data card view mode using buttons to represent the values. Support for ValueButton is introd
 
Use case details

 ValueButton handler is now supported in list view and data card view,
 Static values with number, decimal, date and string are supported.
 When an attribute has no value, this handler shows buttons that can be configured earlier in metadata section.
 Clicking a button sets the attribute to corresponding value.
 When attribute has value, it is shown instead of buttons.
 When a viewer has no permission to edit attribute, buttons are not visible.
