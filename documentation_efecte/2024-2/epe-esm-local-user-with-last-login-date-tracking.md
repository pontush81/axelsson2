Problem statement 
Currently Efecte Admins cannot know activity of the local users from the ESM UI. Only way is to check ESM’s security log. EPE can already create local users via workflow and update those, so we should also monitor are those in use or not in ESM. 
 
Short description
EPE to store user last login date to ESM. EPE can record user login to ESM UI in date and time format. Part of ESM local user lifecycle improvements. 
 
Use case details
The EPE enables easy way for Admins to define are we storing local users last login timestamp or not and where.
