ESS2 comments require using the Comment handler, which is not yet supported in the new Agent UI. Commenting allows support agents to interact efficiently with users reporting issues.
Short description
Bringing support for Comment handler, allowing two-way ESM-ESS2 commenting.
Use case details

 Agents can send a comment to ESS2.
 Agents can read comments from ESS2 based on their user permissions.
 Agents can see which comments are unread.
 The same UI is used as with ESS comments (Worklog handler).
 Commenting is supported in the mobile UI.
