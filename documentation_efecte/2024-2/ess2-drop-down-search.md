Problem statement
End users need help to search for the value they need from drop-down form fields with hundreds of values.
 
Short description
The end users shall have a filtering search to navigate quickly to the value they seek.
 
Use case details
As an end user, I want to be able to search for a specific value in drop-down or multi-value drop-down fields so that I can quickly find or filter down to the value that I am looking for.
 
AC:

 all drop-down and multi-select drop-down fields shall have search functionality to narrow down the results
 search shall be available for the following data types on ESS2 UI: static values, custom values and reference values
 filtering shall start when end-user has given three letters
 search from visible value (datacard name)
