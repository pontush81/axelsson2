ContentsProblem statementShort descriptionUse case detailsProblem statement
It has not been possible to view and edit decimal datatype attributes in the new UI.
 
Short description
Decimal datatype support is brought to the new agent UI, allowing the view of existing values and editing them.
 
Use case details

 It will be possible to see and edit decimal numbers like in the old UI.
 Shown number of decimals will be based on the configuration.
 Using comma and period as the separator will be possibe.
 Decimal number viewing and editing is available based on the user’s permissions to the target attribute.
