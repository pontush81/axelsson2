Data card transform has not been supported in the new Agent UI.
Short description
Datacard transform provides powerful capabilities to create new data cards from an existing datacard and link the datacards between each other. This existing feature is now supported in the new UI.
Use case details

 Support for adding new data card using Transform is now added in the new Agent UI
 The feature behaves the same as in the Classic UI
