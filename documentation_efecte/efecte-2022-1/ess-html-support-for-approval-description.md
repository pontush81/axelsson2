 User Story:  
 As a manager,  
 I want to see details in approval description with HTML formatting,  
 so that reading the information is easier.  
  
 With this feature, we are introducing support for following HTML tags from ESM's additional information in ESS approval view:  
 font,b,i,u,s,tt,blink,marquee,table,tr,td,bgcolor,link,href,a,br,div,em,img,li,ol,p,span,strong,ul,br
