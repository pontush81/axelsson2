 As an IGA Administrator, I want to assign manual access right requests from Efecte ESM to another Efecte ESM installation.  
 As an ITSM Administrator, I want to assign tickets to vendors or other partners that are using also Efecte ESM.  
 As an Admin of Efecte's Provisioning Engine tool (EPE), I want to use connector from Efecte ESM to another Efecte ESM installation.  
Background information: 
What? 
 
 New Efecte connector to EPE 
 
How? 
 
 Administrators are able configure connection to Efecte ESM using the EPE admin UI. 
   
   Processes can be run event-based triggered by Visual Workflow Automation 
   Any data card can be provisioned to another ESM by using event-based provisioning task, workflow and orchestration node 
   Provisioning happens only from one ESM to another 
   All type of attributes from the data card can be provisioned, also attachments 
    
 
Why? 
 
 EPE connector catalog must be comprehensive for customers 
 Efecte ESM connector support has been requested by multiple customers 
 
To whom? 
 
 EPE Administrators 
 EPE Customers
