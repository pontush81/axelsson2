 User story:  
 
 As an admin user, I want to see details about VWA execution.  
 
 ACs:  
 
 The following information should be included in the log 
   
   Log level 
   Date and time 
   Data card ID and name 
   Node ID and name 
   Workflow ID, name, template ID and wrapper ID 
    
 A log entry should be added in the following scenarios. 
   
   Datacard enters a workflow 
     
     Workflow ID 
     Workflow name 
     template ID 
     wrapper ID 
      
   Datacard enters a new node 
     
     Node ID 
     Node name 
     Node details 
      
   Datacard is paused due to a wait node or timer
