 User story: As a customer I want to do provisioning / de-provisioning to 389 LDAP Directory server  
  ACs:  
 
 Efecte Provisioning Engine (EPE) must be able to read (scheduled) and write (event-based) data from and to 389 LDAP Directory Server 
 Administrators are able configure connection to 389 LDAP Directory Server using the admin UI 
   
   Processes can be scheduled in ESM Provisioning Tasks UI 
   Processes can be run manually from ESM Provisioning UI 
   Processes can be run event-based triggered by Visual Workflow Automation 
    
 
 Following event-based provisioning capabilities will be available:  
 
 Add User to Group 
 Create Group 
 Create User 
 Delete Group 
 Delete User 
 Read User's data 
 Remove User from Group 
 Run provisioning task 
 Verify Group 
 Verify User
