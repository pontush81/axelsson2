 AC:  
 - Instead of showing "ESM 2018.3 © Efecte Corp. 1999-2017" tab of the browser should show "Efecte | <primary attribute value> [<ID>]" in detail data card view - if no attribute with ID exists in the currently open data card then no ID shall be displayed - Instead of showing "ESM 2018.3 © Efecte Corp. 1999-2017" tab of the browser should show "Efecte | <view bookmark name>" in the user's preferred language in list, calendar, or graph view- - - Instead of showing "ESM 2018.3 © Efecte Corp. 1999-2017" tab of the browser should show "Efecte | Administration" in Admin UI - Instead of showing "ESM 2018.3 © Efecte Corp. 1999-2017" tab of the browser should show "Efecte | Background Data" in Background Data UI - Instead of showing "ESM 2018.3 © Efecte Corp. 1999-2017" tab of the browser should show "Efecte | Home" on Homepage if no content is open
