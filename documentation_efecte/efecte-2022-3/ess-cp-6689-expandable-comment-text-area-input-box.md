 As an end user, I want to be able to easily browse through my comment, to be able to review the content before sending.  
  
 ACs:  
 
 Input box for comment shall be changed to text area 
 Default size of the text area shall be for one row (height) 
 Text area shall be expandable at the bottom right corner 
 Pressing Enter shall result in a line break instead of submitting the comment 
 Long text shall not be hidden 
 Long text shall not overflow outside the input box
