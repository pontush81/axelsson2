           Fixed
        

        
    
 The following bug (CP-6331) has been identified in release ESM 2021.2:  
 If the Virtual Coach is not working after the upgrade to ESM release 2021.2, navigate to the Virtual Coach settings under System Settings and 1. Press "Edit" and then "Save". 2. Press Enabled No -> Save. Then press Enabled yes -> Save.  
 This will fix the issue.
