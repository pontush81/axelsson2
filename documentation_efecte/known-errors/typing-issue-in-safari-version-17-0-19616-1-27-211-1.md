The following issue has been identified when using Efecte ESM version 2023.3 with Safari:
CP-7819: EntityStateMail typing issue after line break is added in Safari browser
When writing an email and pressing Enter to create a line break, typing in text is only working after clicking the space bar. 
This happens with Safari version 17.0 (19616.1.27.211.1).
Workaround: Use Shift + Enter to create a line break.
