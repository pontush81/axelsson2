 The following bug (CP-6335) has been identified in release ESM 2021.2:  
 When embedding an attachment (for instance a screenshot) in an email which has at least one white space in the file name, the picture is not displayed correctly.  
 Workaround:  
 Avoid using white spaces in file names.
